PCAT – Fotoğraf Yükleme Uygulaması
Demo : https://pcat-app-8hsr.onrender.com

Bu proje, Node.js ve Express.js kullanılarak geliştirilen basit bir fotoğraf yükleme uygulamasıdır. Amaç, backend geliştirme sürecinde sık kullanılan araç ve kavramları pratik bir proje üzerinden öğrenmektir.

Projede Express ile server kurulumu, EJS template engine ile dinamik sayfa render etme, MongoDB ve Mongoose ile veritabanı işlemleri yapılmıştır. Uygulama MVC mimarisi kullanılarak yapılandırılmış, CRUD işlemleri ve pagination (sayfalama) mantığı uygulanmıştır. Geliştirme sürecinde Nodemon ve Prettier kullanılmış, proje GitHub üzerinden yönetilmiştir. Veritabanı MongoDB Atlas üzerinde tutulmuş ve uygulama Heroku’ya deploy edilmiştir.
